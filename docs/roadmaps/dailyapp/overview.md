# はじめに

## デザインモック

[Figma](https://www.figma.com/file/q3zbcZ9qrMRDTUi72QFEv2/DailyApp?type=design&node-id=502%3A38&mode=design&t=CcFyqT1oWOuvTLvd-1)

## アプリケーション仕様

- 認証機能がある
- 日記を作成できる
  - タイトルを入力できる
  - 本文を入力できる
  - 今日の調子（5段階）を登録できる
- 日記一覧や詳細を閲覧できる
- 過去の調子をグラフ形式で閲覧できる←この辺りがフィットネス感
